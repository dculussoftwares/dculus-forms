What AI feature should we build next?

The AI chat drawer already handles field CRUD, page management, quick chips, and streaming. Here are the most impactful next directions.

A

Generate Form from Description

User types "Create a job application form with resume upload" and the AI builds the entire form from scratch — pages, fields, labels, validation — in one shot. Replaces blank-canvas friction.

High impact New tool: generateForm Shows on empty canvas
B

AI Response Analyser

After responses come in, the AI reads them and surfaces insights: drop-off fields, most-skipped questions, common free-text themes, completion rate by device. Lives in the Responses / Analytics tab.

High value post-launch New: AI Insights panel Reads response data
C

Add / Remove Page Tools

The AI can rename and reorder pages but can't add or delete them. Adding addPage and removePage tools closes this gap — the AI becomes fully capable of restructuring multi-page forms end-to-end.

Closes existing gap New: addPage / removePage Low effort, high leverage
D

Smart Field Suggestions Sidebar

As the user builds a form, the AI watches the existing fields and proactively suggests the next 3 logical fields to add ("You have Name + Email — add Phone, Company, Message?"). One click to insert.

Always-on guidance Suggestion panel in builder Debounced AI calls
