Proposed AI Chat Feature Ideas

Select all that interest you — we'll prioritise and design the ones you pick. Add your own ideas in the terminal.

1

Field highlight on add/edit

When the AI adds or modifies a field, the canvas auto-scrolls and highlights that field with a pulse ring — so you can see exactly what changed without hunting for it.

2

Live tool-call status in chat

Instead of a generic spinner, show real-time inline labels: "Reading form structure…", "Adding field…", "Updating layout…". Users know exactly what step the AI is on.

3

Token usage meter

A small progress bar in the drawer header showing % of monthly AI quota used. Clicking it opens a tooltip with the exact count and plan limit. Prevents surprise 402 errors.

4

Context-aware quick chips

Quick-action chips that change based on form state — empty form gets "Generate from description", filled form gets "Analyse & suggest improvements", multi-page form gets "Reorganise pages".

5

AI-suggested field validation

New tool: the AI can read a field label and suggest sensible validation rules (e.g. "Phone number" → maxLength 15, "Age" → min 0 max 120). Shown as a chip the user can accept or dismiss.

6

Per-message undo badge

Each assistant message that made mutations shows an inline "Undo this" badge. Clicking it reverses exactly the changes from that turn, not all Y.js history.

7

Change summary card

After each AI response that modifies the form, show a compact diff card listing "+ Added", "~ Updated", "− Removed" fields — so the user has a changelog without reading the full message.

8

Page-switch command

The AI can navigate the builder to a different page mid-conversation. "Go to page 2" switches the canvas view — useful when the AI is editing a page the user isn't currently on.

9

"Remix this form" command

Say "Make this a job application form instead" — the AI analyses the current structure and transforms field types/labels to fit the new purpose, reusing as much structure as possible.

10

Bulk field operations

New tool: AI can apply changes across all fields matching a pattern — "Add a hint to every text field", "Make all date fields optional", "Rename all labels to sentence case".
